# ポートフォリオサイト制作
お問い合わせ機能付きの動的な自己紹介サイト


## 要件
### クリア条件
- Next.jsであること
- URL公開
- ポートフォリオ機能
  
### 技術スタック
- Next.js
- Vercel / Firebase
- Tailwind CSS
  
### 画面
- ホーム画面:自己紹介と制作物を掲示
- 閲覧者がお問い合わせ画面からフォームで問い合わせる
- システムが専用URLを発行
- 管理者画面から裏側でお問い合わせを承認/拒否
- 専用URLでチャットで会話
