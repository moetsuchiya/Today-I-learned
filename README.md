# メモ
- feat: 新しい機能
- fix: バグの修正
- docs: ドキュメントのみの変更
- style: 空白、フォーマット、セミコロン追加など
- refactor: 仕様に影響がないコード改善(リファクタ)
- perf: パフォーマンス向上関連
- test: テスト関連
- chore: ビルド、補助ツール、ライブラリ関連



# ポートフォリオサイト制作
お問い合わせ機能付きの動的な自己紹介サイト


## 要件
### クリア条件
- Next.jsであること
- URL公開
- ポートフォリオ機能
  
### 技術スタック
- Next.js
- Vercel / Firebase
- Tailwind CSS
  
### 画面
- ホーム画面: 自己紹介と制作物を掲示
- お問い合わせ画面: 閲覧者がお問い合わせ画面からフォームで問い合わせる & システムが専用URLを発行
- 管理者画面: 管理者が裏側でお問い合わせを承認/拒否
- お問い合わせチャット画面: 専用URLでチャットで会話
